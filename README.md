# やり方

１、ZenMate VPN .crxのファイルを押す。

２、View rawを押し、保存する。

３，chrome://extensions/
ココに行く。

４、ファイルに保存されたZenMate VPNを ３ の場所にコピーする

５、拡張機能をONにする(sign upしてから)

６、https://www.youtube.com
に行く

７、完璧☆
